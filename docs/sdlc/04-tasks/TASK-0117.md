# TASK-0117: Tenant Call-Center Statistics and Quota Usage API

## Outcome

Expose a tenant-scoped API that returns date-filtered call-center statistics and quota usage from the analytics and entitlement sources.

## Acceptance criteria

1. `GET /api/tenant/call-center/statistics` supports `start_date` and `end_date` in `YYYY-MM-DD` format.
2. Omitting both dates resolves the range to today in the tenant deployment timezone.
3. The response includes total sessions, chat/voice counts, duration or minutes, avatar breakdown, and quota used/remaining.
4. The endpoint requires the tenant role and derives tenant scope from the authenticated context.
5. Invalid ranges, unavailable analytics, and unauthorized access return stable error codes without cross-tenant metadata.
6. Existing `/api/tenant/usage` and `/api/tenant/satisfaction/statistics` contracts remain backward compatible.
