# TASK-0146: @monti/embed-core

## Outcome

Ship a framework-agnostic TypeScript package that resolves public embed config, builds the iframe URL, mounts floating or inline widgets, and supports open/close/destroy without leaking listeners or orphan iframes.

## Acceptance criteria

1. `MontiEmbed` mounts with `embedKey` + `apiBase`; optional `parentOrigin`, `position`, `agentId`, `theme`, `locale`, `container`.
2. Pre-resolve maps `embed_not_found` / `embed_disabled` / `origin_not_allowed` to clear `error` events.
3. Unit tests cover open/close/destroy cleanup and bad-key error.
4. Vanilla `monti-embed.js` remains the zero-dep path (no breaking change).

## Notes

Package: `packages/embed-core`. Tests: `packages/embed-core/test/lifecycle.test.mjs`.
