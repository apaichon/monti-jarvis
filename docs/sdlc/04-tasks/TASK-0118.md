# TASK-0118: Tenant Call-Center Dashboard and Quota Usage UI

## Outcome

Add a tenant dashboard that makes call activity and quota consumption easy to scan and filter.

## Acceptance criteria

1. Add a tenant dashboard route using the existing tenant navigation and visual conventions.
2. The initial view selects today and provides start-date and end-date controls.
3. The layout shows summary metrics, channel and avatar breakdowns, call minutes, and quota used/remaining.
4. Loading, empty, invalid-range, unauthorized, and analytics-error states are explicit and non-overlapping on desktop and mobile.
5. The UI uses the existing API client/auth/session patterns and does not accept tenant scope from user-editable input.
6. Tenant frontend build and focused dashboard checks pass.
