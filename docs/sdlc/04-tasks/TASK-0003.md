# TASK-0003: LiveKit Room/Token API + Join Flow

## Outcome

Go API creates call sessions, issues LiveKit JWT tokens, and Svelte client joins room with microphone.

## Acceptance Criteria

- `POST /api/calls` creates session + room name
- `POST /api/calls/{id}/token` returns JWT + LiveKit URL
- `internal/lktoken` generates access tokens
- Customer UI connects via `livekit-client` and enables mic
