# TASK-0122: Tenant System Performance Monitoring Dashboard

## Outcome

Give tenant administrators a clear operational view of the current service path and analytics freshness.

## Acceptance criteria

1. Add a tenant monitoring route and navigation entry using the existing tenant-web design language.
2. Show overall status, component status, latency, last checked time, and analytics freshness without raw infrastructure details.
3. Implement loading, operational, degraded, unavailable, stale, unauthorized, and retry states.
