# TASK-0138: Platform call-center statistics dashboard

## Outcome

Add a platform-admin dashboard for overall and by-tenant call-center statistics.

## Acceptance criteria

1. Add a discoverable `/admin/call-center` route using the existing platform-admin navigation and visual system.
2. Render completed conversations, total talk time, average conversation, channel/avatar/satisfaction summaries, and package usage.
3. Provide start/end date controls, a Today action, bounded tenant pagination, loading, empty, stale, unavailable, retry, and session-expired states.
4. Keep labels and values readable at desktop and mobile widths without overlap or horizontal scrolling.
5. Preserve the existing platform monitoring, audit-log, and tenant-admin navigation behavior.
