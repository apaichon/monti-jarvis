# TASK-0063: Public Embed API

## Outcome

Public resolve endpoint and tenant-admin CRUD for embed config; origin checks and security headers.

## Acceptance Criteria

1. `GET /api/public/embed/{embed_key}` — no auth:
   - 200: `{ tenant_id, slug, name, embed_key, default_agent_id, agents? }` when enabled
   - 404: unknown / disabled
2. Origin check: if `allowed_origins` non-empty, require `Origin` or `Referer` host match; else 403 `origin_not_allowed`.
3. Tenant admin (active):
   - `GET /api/tenant/embed`
   - `PUT /api/tenant/embed` (enabled, allowed_origins, default_agent_id)
   - `POST /api/tenant/embed/rotate-key` → new `embed_key`
4. Lazy-create embed config on first GET if missing (key generated).
5. CORS on public embed routes respects allowlist (or `*` when empty + dev).
6. Unit/handler tests: resolve ok, disabled, bad origin, rotate invalidates old key.

## Verification

```bash

curl -s http://localhost:8091/api/public/embed/$KEY | jq .
```
