# TASK-0151: Tenant Theme editor

## Outcome

Tenant admins edit **brand name, logo, subtitle**, and **all color tokens** with a live preview of the caller/embed chrome, then draft/publish/reset.

## Acceptance criteria

1. Route `/tenant/theme` (or Settings → Theme) for active tenant_admin.
2. Brand section: name, subtitle, logo upload + preview.
3. Color section: preset + pickers for every required token.
4. Live preview approximates screenshot: header (logo/name/sub), primary CTA, chat bubble, Send.
5. Save draft / Publish / Reset wired to APIs; contrast warnings + confirm on publish.
6. `npm run check` clean for tenant-web.

## Notes

Preview must show that **any** brand field and **any** color can change — not colors alone.
