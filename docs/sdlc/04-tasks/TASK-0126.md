# TASK-0126: Typed mobile SDK core and adapters

## Outcome

Ship a dependency-light TypeScript SDK core with injectable storage, fetch, WebSocket, and platform notification adapters.

## Acceptance criteria

1. SDK exposes OTP, bootstrap, token refresh, call lifecycle, reconnect, transcript, end, and 1-5 star rating operations.
2. Provider credentials and tenant switching are impossible through the public SDK surface.
3. Browser, React Native, and native host adapters can supply platform primitives without forking call logic.
4. The SDK is buildable independently from the customer web application.

## Verification

- Added `packages/monti-mobile-sdk` with typed bootstrap, OTP, token refresh, call, WebSocket, transcript, end, and star-rating APIs.
- SDK build passed with standalone `tsc -p packages/monti-mobile-sdk/tsconfig.json`.
