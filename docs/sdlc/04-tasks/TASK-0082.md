# TASK-0082: Tier Schema

## Outcome

Postgres tables for tenant-scoped customer tiers and groups; ensure-schema on startup.

## Acceptance Criteria

1. `callcenter.customer_tiers`:
   - `id` text PK  
   - `tenant_id` text FK → tenants  
   - `name`, `slug` text  
   - `priority` int default 0  
   - `description` text  
   - `default_agent_id` text nullable  
   - `ai_reply_locale` text default ``  
   - `max_minutes_per_call` int default 0  
   - `max_call_minutes_per_day` int default 0  
   - `active` boolean default true  
   - audit columns  
   - UNIQUE (tenant_id, slug)  
2. `callcenter.customer_groups`:
   - `id` text PK, `tenant_id`, `name`, `slug`, `description`, audit  
   - UNIQUE (tenant_id, slug)  
3. No customer assignment table yet (S19+).  
4. Document no new env vars required.
