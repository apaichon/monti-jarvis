# TASK-0116: ClickHouse Tenant Call-Center Analytics Projection

## Outcome

Create the tenant-scoped ClickHouse analytics contract and projection path needed by the Sprint 25 dashboard.

## Acceptance criteria

1. ClickHouse schema and migration/bootstrap are documented and reproducible for the Monti Jarvis database.
2. Facts include tenant id, call/session id, channel, avatar id, started/ended timestamps, and billable duration or minutes.
3. Projection writes are idempotent for retries and duplicate source events.
4. Tenant and customer contact data, transcript text, and audio content are excluded from analytics facts.
5. A backfill or replay path is documented for existing Sprint 22-24 conversation records.
6. Projection and query indexes support tenant plus date-range filtering.
