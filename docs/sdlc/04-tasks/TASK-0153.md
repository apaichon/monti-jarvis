# TASK-0153: Theme infrastructure pricing and contract discounts

## Outcome

Document Thai-baht infrastructure estimates, BYOK package prices, and prepaid
six-month and one-year contract discounts for the theme-enabled platform.

## Acceptance criteria

1. The design spec records the EUR-to-THB planning exchange rate and source.
2. The design spec records machine counts and monthly infrastructure estimates
   for the documented registered-user scale assumptions.
3. BYOK package prices include monthly, six-month, and one-year totals.
4. Gemini provider charges remain explicitly excluded from Monti pricing.
5. Contract assumptions, rounding, tax, and Contabo price volatility are stated.
