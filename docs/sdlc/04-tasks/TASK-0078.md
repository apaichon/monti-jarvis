# TASK-0078: Preview APIs

## Outcome

Tenant-admin-only endpoints (or query flags) that run chat/voice under JWT tenant with **preview** semantics.

## Acceptance Criteria

1. Auth: `RequireTenantAdminActive`; tenant id from JWT only.  
2. Endpoints (preferred):

   | Method | Path | Purpose |
   | --- | --- | --- |
   | POST | `/api/tenant/preview/chat` | Text chat as preview (RAG + locale) |
   | GET | `/ws/tenant/preview/voice` | Voice WS preview (or `/ws/voice?preview=1` with JWT) |

3. Preview **skips** `AddCallMinutes` / daily incr / monthly check for minutes; still applies:  
   - rate limit (`AllowRate`)  
   - soft concurrent preview cap  
   - RAG + AI locale hint  
4. Production `/api/chat` and `/ws/voice` unchanged for minute accounting.  
5. Errors: clear codes for concurrent preview cap, inactive tenant.  
6. Unit/integration tests: preview does not call monthly incr; prod path still does.
