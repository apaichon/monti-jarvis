# TASK-0137: Platform call-center statistics API

## Outcome

Expose a protected, date-filtered platform statistics endpoint with aggregate KPIs and a bounded tenant breakdown.

## Acceptance criteria

1. Only platform administrators can read the endpoint.
2. Start/end dates default to today and reject invalid or reversed ranges with a safe validation error.
3. Tenant pagination is bounded and returns stable totals, offsets, and freshness metadata.
4. Response fields are allowlisted and contain no raw ClickHouse/provider errors or customer-level data.
5. ClickHouse unavailable and stale states map to stable API errors/status fields without breaking tenant routes.
