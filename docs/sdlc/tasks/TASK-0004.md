# TASK-0004: Call Session + NATS Events

## Outcome

Call lifecycle persisted in Postgres/Redis 8 and published on NATS.

## Acceptance Criteria

- Tables `call_sessions`, `call_turns` in schema `callcenter`
- Redis 8 active call keys `monti_jarvis:call:active:{id}`
- NATS publishes `call.started`, `call.ended`, `call.turn.created`
- `POST /api/calls/{id}/end` sets `ended_at`
