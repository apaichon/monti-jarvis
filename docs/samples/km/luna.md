# Luna — Technical Support Knowledge

## First step for outages
Ask whether the issue affects one device or all devices. If all devices are offline, check the status page at status.monti.example.

## App login loop
Clear browser cache, disable VPN, and retry. If the loop persists, collect the error code shown on screen.

## Microphone on calls
Ensure the browser has microphone permission and no other app is using the mic. Reload the page after granting permission.

## Supported browsers
Latest Chrome, Edge, and Safari are supported for voice calls. Firefox voice is experimental.
