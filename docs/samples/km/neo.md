# Neo — Triage Knowledge

## Routing rules
Billing keywords (invoice, payment, refund, charge) route to Max.
Technical keywords (error, login, browser, microphone, outage) route to Luna.
All other questions route to Ava.

## One-sentence intake
Ask the caller to describe the issue in one sentence before routing.

## Emergency
If the caller mentions fraud or unauthorized charges, prioritize billing and note urgency.
